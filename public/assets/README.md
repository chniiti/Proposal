# Assets Folder

Place your files here:

1. **Profile Photo**: `profile-photo.jpg` (or .png)
   - Recommended size: 200x200px minimum
   - Will be displayed as a circular image in the header

2. **CV**: `Bilel-Chniti-CV.pdf`
   - Your resume/CV document

3. **Proposal**: `Bilel-Chniti-Executive-Proposal.pdf`
   - Your business proposal document

All files in this folder will be accessible at `/assets/filename.ext`
